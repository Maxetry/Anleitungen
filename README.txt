Ihr könnt Python entweder in Anaconda oder VSCode schreiben.
Die Installation von Anaconda ist auf dem Heimrechner deutlich einfacher!!!


Wenn ihr VSCode mit C++, dem MXChip und allem anderen installieren wollt, benutzt Anleitung "Anleitung_VSCode_CPP_MXChip"

Wenn Ihr VSCode nur für Python installieren wollt, benutzt bitte die Anleitung "Anleitung_VS_Code" und installiert die "Python" Extension.

Für Anaconda gibt es die Anleitung "Anleitung_Anaconda_Python".
